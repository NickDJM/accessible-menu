Hello Accessible Menu!

This is a demo and manual testing application for accessible-menu.

This app allows you to swamp between one, two, and three levels of navigation for all supported types of menus provided by accessible-menu.

You can also swap between light and dark mode for ease of use on you if you find white screens too bright.

For more in-depth information about how accessible-menu works, please read the documentation.
